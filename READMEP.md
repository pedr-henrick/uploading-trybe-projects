## ⬇️ Instalando dependências

Backend

  ```bash
  cd api/ 
  npm install
  ``` 
Frontend

  ```bash
  cd src/
  npm install
  ``` 

## ⚡ Executando a aplicação

Para rodar o back-end:

  ```
  cd api/ && npm start
  ```
Para rodar o front-end:

  ```
  cd src/ && npm start
  ```

## 🧪 Executando os testes

Para rodar todos os testes:

  ```
  npm test
  ```
